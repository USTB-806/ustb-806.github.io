    添加文章

    目录

    若需要在 806 官网上添加文章（新闻、博客、……)，请严格遵循以下步骤：

    1. 告知所在部门的负责人并经过其同意

    2. fork 本仓库，配置环境并按照格式编写文章（请注意不同类型的文章的编写要求）

      • 添加新闻请在 /pages/_news 目录下按照年份分类添加

      • 添加博客请在 /pages/_blogs 目录下添加

    3. 格式化代码（否则将无法通过代码审查）

    4. 在本网站的 GitHub 仓库提交一个 pull request

    5. 待负责人审查通过后合并进 main 分支，完成更新
